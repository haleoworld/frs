Cloud sync (GitHub Gist)

Auto-saves every change to a private GitHub Gist you own. Every save becomes a versioned revision, so you can roll back to an earlier copy if anything ever goes wrong. Leave the token blank to stay offline (data lives only in this browser).

First-time setup instructions (2 minutes)
  1. Open github.com/settings/tokens/new (GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)).
  2. Under Select scopes, tick only gist — nothing else.
  3. Pick an expiry (or "No expiration" to avoid redoing this), then Generate token and copy it (it starts with ghp_).
  4. Paste the token above, leave Gist ID blank, and click Save & test. The app creates a new private gist and saves your data; its ID appears just above.
  5. On any other device: open this same page, paste the same token and that Gist ID, then click Pull from cloud.

Restore an older copy: open the gist on GitHub, click Revisions, choose an earlier save, copy its raw JSON, and paste it into Settings → Import JSON. Your token stays in this browser only — it is never written into the gist.
